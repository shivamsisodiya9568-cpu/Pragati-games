## 🌟 About

**Pragati Kids Games** is a premium, responsive, colorful mini-games website built for children.  
It includes 6 fun, educational, and interactive games — all free, safe, and browser-based.

Designed with a **premium modern UI** featuring:
- Glassmorphism card design
- Animated gradient backgrounds
- Smooth scroll reveal animations
- Floating bubble particles
- Confetti win celebrations
- Toast notifications
- Fully responsive layout (mobile, tablet, desktop)

---

## 🎮 Games Included

| # | Game | Description | Score System |
|---|------|-------------|--------------|
| 1 | **Tic Tac Toe** | Classic 2-player X & O strategy game | Win counter for X and O |
| 2 | **Memory Cards** | Flip and match emoji pairs | Move counter |
| 3 | **Number Guessing** | Guess the secret number (1–100) in 7 tries | Win counter |
| 4 | **Rock Paper Scissors** | Battle the computer — first to 5 wins! | Score for You vs CPU |
| 5 | **Click the Ball** | Click balls as fast as possible in 30 seconds | Score + Best score |
| 6 | **Math Quiz** | Answer 10 math questions (add, subtract, multiply) | Score out of 10 |

## 📁 Project Structure

```

│
├── index.html       # Main HTML file — all sections and game markup
├── style.css        # Complete CSS — animations, glassmorphism, responsive layout
├── script.js        # All game logic + UI interactions
└── README.md        # Project documentation (this file)
```

## 🚀 How to Run

### Option 1: Open Directly
Simply double-click `index.html` to open it in your browser — no server needed!

### Option 2: Live Server (Recommended)
If you have VS Code with the Live Server extension:
1. Open the project folder in VS Code
2. Right-click `index.html`
3. Select **"Open with Live Server"**

### Option 3: Any Web Server
Place the 3 files (`index.html`, `style.css`, `script.js`) in any web server directory and open in browser.

## 🛠️ Technologies Used
